Pure JavaScript : JavaScript olduğu haliyle yani herhangi bir kütüphane ve framework eklemeden kullandığımız temel düzey JavaScripttir.

JavaScript Selectors (Seçiciler) : HTML elementlerini JavaScript yoluyla seçmemize yarar. Bunları seçip işlem yapabilir ya da değişkenlere atayabiliriz.

JavaScript DOM (Document Object Model)

JavaScript Events : Sayfa içinde interaktif yani kullanıcının etkileşime geçebileceği aksiyonları yönetmek için kullandığımız olaylar.
